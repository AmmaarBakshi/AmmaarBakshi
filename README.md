### 🔧 Featured Projects

**[git-recall](https://github.com/AMMAAR-IC/git-recall)** `C`

Personal git standup tool written in pure C. See what you committed by day, week, month, or year. No dependencies — just git. Ships on both **winget** and an **Ubuntu PPA**.

```bash
$ git-recall --week
$ git-recall --month -2        # last 2 months
$ git-recall --day > recap.txt # pipe to file
```

[![APT PPA](https://img.shields.io/badge/PPA-ammaar--apt%2Fgit--recall-E95420?style=flat-square&logo=ubuntu&logoColor=white)](https://launchpad.net/~ammaar-apt/+archive/ubuntu/git-recall)

**[paimon](https://github.com/AMMAAR-IC/paimon)** `Rust`

Keyboard-driven Rust TUI for injecting code snippets and project structures into your working directory instantly. Snippets are embedded inside the binary at compile time — no internet, no Python, nothing extra needed.

```bash
$ paimon --code -ls
# ↑↓ navigate   ENTER select   ← back   q quit
```

[![winget](https://img.shields.io/badge/winget-Ammaar.Paimon-0078D4?style=flat-square&logo=windows&logoColor=white)](https://github.com/microsoft/winget-pkgs)
[![npm](https://img.shields.io/badge/npm-paimon--npm-CB3837?style=flat-square&logo=npm&logoColor=white)](https://www.npmjs.com/package/paimon-npm)

**[letx](https://github.com/AMMAAR-IC/letx)** `Python`

Modular Python CLI toolkit. Debug smarter with plain-English error explanations, get fix suggestions, and clean your codebase — all from the terminal.

```bash
$ letxDebug -a my_script.py    # run + explain + suggest fix
$ letxFix -rm -cmt src/        # strip all comments from a folder
```

[![PyPI](https://img.shields.io/badge/PyPI-letx-3775A9?style=flat-square&logo=pypi&logoColor=white)](https://pypi.org/project/letx/)
